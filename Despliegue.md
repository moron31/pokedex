# Despliegue de PokeDex en Azure

## Plataforma usada
Microsoft Azure App Service

## Pasos realizados

1. Descarga del proyecto
2. Instalación de Node.js
3. npm install
4. npm start
5. npm run build
6. Creación de App Service en Azure
7. Subida de archivos a wwwroot
8. Verificación final

## Resultado

Aplicación desplegada correctamente.
