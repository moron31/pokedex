## Descripción del Proyecto

PokeDex es una aplicación web desarrollada con Angular que consume la API pública PokéAPI para mostrar información detallada de distintos Pokémon de forma dinámica, visual e interactiva.

La aplicación permite consultar datos como nombre, imagen, tipo, estadísticas y características generales de cada Pokémon mediante una interfaz moderna y amigable para el usuario.

El objetivo principal del laboratorio fue ejecutar el proyecto localmente y posteriormente desplegarlo en la nube utilizando Microsoft Azure.

---

## Proceso Realizado

### 1. Descarga del Proyecto

Se descargó el archivo fuente del laboratorio y se descomprimió correctamente en el equipo local.

### 2. Instalación de Dependencias

Ubicados dentro de la carpeta del proyecto, se ejecutó el siguiente comando:

npm install

Este comando instaló todas las librerías necesarias para Angular.

### 3. Ejecución Local

Para probar la aplicación localmente se utilizó:

npm start

La aplicación quedó disponible en:

http://localhost:4200/

### 4. Compilación del Proyecto

Para generar la versión lista para producción se ejecutó:

npm run build

Esto creó la carpeta `dist/pokedex-angular` con los archivos finales del proyecto.

### 5. Despliegue en Azure

Se creó un App Service en Microsoft Azure y se cargaron los archivos compilados para su publicación en internet.

### 6. Verificación Final

Se comprobó que la aplicación quedó accesible públicamente y funcionando sin errores.
